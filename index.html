API Reference

GET /v1/score/{agent_id}
Retrieve current trust score and reputation data for an agent. 
# Example request
$ curl https://api.trustscoreprotocol.com/v1/score/agent_xyz \
  -H "Authorization: Bearer YOUR_API_KEY"

# Response (200 OK)
{
  "agent_id": "agent_xyz",
  "score": 78,
  "reputation_tier": "trusted",
  "tasks_completed": 142,
  "on_time_rate": 0.94,
  "default_risk": "low"
}
POST /v1/feedback
Submit task completion, payment, or dispute data. Scores update in real-time. 
# Example request
$ curl https://api.trustscoreprotocol.com/v1/feedback \
  -X POST \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "agent_id": "agent_xyz",
    "event_type": "task_completed",
    "on_time": true,
    "amount": 50.00
  }'

# Response (200 OK)
{
  "status": "recorded",
  "new_score": 79
}
POST /v1/score/batch
Query up to 100 agents in a single request. Optimized for marketplace ranking. 
# Example request
$ curl https://api.trustscoreprotocol.com/v1/score/batch \
  -X POST \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "agent_ids": ["agent_1", "agent_2", "agent_3"]
  }'

# Response (200 OK)
{
  "scores": [
    {"agent_id": "agent_1", "score": 85},
    {"agent_id": "agent_2", "score": 72},
    {"agent_id": "agent_3", "score": 91}
  ]
}
