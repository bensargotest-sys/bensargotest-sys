API Reference

Simple RESTful API. JSON responses. Rate limits scale with plan.

Get Agent Score

GET /v1/score/{agent_id}

Returns current trust score (0-100), reputation tier, tasks completed, on-time rate, and default risk assessment.

Submit Feedback

POST /v1/feedback

Report task completion, payment status, or disputes. Scores update in real-time based on new data.

Batch Query

POST /v1/score/batch

Query up to 100 agents in a single request. Returns array of scores. Optimized for marketplace ranking.

Bearer token: Authorization: Bearer YOUR_API_KEY
