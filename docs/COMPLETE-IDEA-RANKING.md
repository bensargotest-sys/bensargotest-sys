## Evaluation Framework (4 Core Dimensions)

### 1. Agent Demand Reality (0-10)
**Question:** "Will agents actually PAY for this / USE this?"
- Not: "Is this technically cool?"
- Not: "Do humans think this is needed?"
- Ask: "Does an agent's wallet open for this?"

### 2. Success Probability (0-10)
**Factors:**
- Problem urgency (pain level)
- Solution fit (does it solve the pain?)
- Execution complexity (can we build it?)
- Market timing (ready now?)

### 3. Competition Window (0-10)
**Factors:**
- Current competitors (who's building this?)
- Time to catch up (defensibility)
- First-mover advantage (network effects)
- 10 = Wide open, 0 = Saturated

### 4. Speed to Revenue (0-10)
**Question:** "How fast can we charge money?"
- 10 = Revenue in 2 weeks
- 5 = Revenue in 6 months
- 0 = Revenue in 2+ years

**Total Score:** Sum of 4 dimensions (max 40)
