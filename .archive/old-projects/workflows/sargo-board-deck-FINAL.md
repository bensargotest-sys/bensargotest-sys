## Delivery Tips

### Pre-Meeting Prep (24 Hours Before)
- Send deck to board 24 hours early (no surprises)
- Prepare for 6 tough questions (see below)
- Practice Act 3 delivery (hardest part - risks + contingency)
- Have backup materials ready (detailed runway spreadsheet, customer contracts)

### During Meeting
- **Tone:** Confident but not cocky. Realistic but not defeated.
- **Body Language:** Lean forward on traction, sit back on risks (shows confidence in mitigations)
- **Pacing:** Spend 40% on Act 1 (proof), 30% on Act 2 (plan), 30% on Act 3 (ask + contingency)
- **Pause:** After Slide 16 (contingency), pause 5 seconds. Let it land. This is the moment of truth.

### Tough Questions (Prepare Answers)

**Q1: "Why should we believe you can 3x volume in 60 days?"**
- A: "We've already grown 20-30% MoM with 1 client. Adding Base + 2nd client = 3x is conservative. If we only hit 2x, story is still 'multi-chain working + client repeatability.'"

**Q2: "What if Base launch delays?"**
- A: "Mitigation: Using audited contracts, partnering with existing DEX, have rollback to testnet demo. Worst case: story becomes 'launching in 2 weeks' instead of 'live now.' Variant still sees momentum."

**Q3: "What if you don't close 2nd client?"**
- A: "Pursuing 5+ prospects, aggressive pilot terms, leveraging Client 1 referrals. Fallback: 3 signed LOIs shows pipeline even if closed revenue delayed. VCs buy the trajectory."

**Q4: "What if Variant passes?"**
- A: "Variant is warm intro, not only shot. Prelude + Charles parallel. 8 batch VCs queued for March. Strategy is FOMO via multiple conversations. If Variant passes, it's data - we pivot pitch."

**Q5: "Can you actually deliver ZKP MVP in 4 weeks?"**
- A: "Yes. Using existing libraries (Circom/Noir), not building from scratch. PoC already working. 4 weeks = integration + security review. Martin is confident. If we slip to mid-March, still time for VC demos."

**Q6: "What happens if we run out of money in April?"**
- A: "Honest answer: We wind down with dignity. 30 days notice to team, product stays live, clients transitioned. No zombie mode. Martin and Daniel have earned honesty - if funding doesn't close by March 31, we make the call April 1. That's the board decision we need today."
